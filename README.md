# Mood-Detector
An AI-powered **Mood Detector** designed to identify human emotions in real-time. Built using **Python**, **OpenCV**, and **Deep Learning**, this project uses a convolutional neural network to analyze facial expressions. It features a clean interface, handles live video feeds, and is optimized for both accuracy and speed in emotion recognition.
 # Mood Detector

An AI-powered application that detects human emotions in real-time using computer vision and deep learning.

 Features
* **Real-time Detection:** Analyzes facial expressions via webcam.
* **Multiple Emotions:** Detects Happiness, Sadness, Anger, Surprise, and more.
* **High Accuracy:** Built with a Convolutional Neural Network (CNN).
* **Fast Inference:** Optimized for low-latency processing.

 Tech Stack
 Language: Python
 Libraries: OpenCV, TensorFlow/Keras, NumPy, Pandas.
 Algorithm: CNN (Convolutional Neural Network).
